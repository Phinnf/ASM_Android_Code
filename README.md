CampusExpense Manager: Mobile Expense Tracking Application
This repository contains the source code for the CampusExpense Manager Mobile Application, an Android application developed to help university students effectively manage their finances and track their spending against set budgets.

The project was undertaken by BudgetWise Solutions as a response to the need for a simple, easy-to-use tool for students living on or off-campus. The core implementation focuses on key features such as secure user accounts and offline expense management.

Technical Stack
Platform: Android
Language: Java.
Database: Android SQLite.
